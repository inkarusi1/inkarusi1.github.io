Insert模式 → 按下键 esc 返回Normal模式

常用功能键记录

前五个最常用

  • i → Insert 模式，按 ESC 回到 Normal 模式.
  • x → 删当前光标所在的一个字符。
  • :wq → 存盘 + 退出 (:w 存盘, :q 退出) （陈皓注：:w 后可以跟文件名）；:q! → 退出不保存，脑残党的福音惹(喜
  • dd → 删除当前行，并把删除的行存到剪贴板里
  • p → 粘贴剪贴板
  • yy → 拷贝当前行

光标的移动：

  • NG → 到第N行，也可以用:N到第N行，这更常用
  • gg → 到第一行
  • G → 到最后一行
  • % → 匹配括号移动（要先把光标移到括号上）

  • * 和 #: 匹配光标当前所在的单词，移动光标到下一个（或上一个）匹配单词（*是下一个，#是上一个）

  • 0 → 到行头

  • ^ → 到本行的第一个非blank字符

  • $ → 到行尾

  • g_ → 到本行最后一个不是blank字符的位置。

  • fa → 到下一个为a的字符处，你也可以fs到下一个为s的字符。

  • t, → 到逗号前的第一个字符。逗号可以变成其它字符。

  • 3fa → 在当前行查找第三个出现的a。【以光标为基点】

  • F 和 T → 和 f 和 t 一样，只不过是相反方向。

  • w → 到下一个单词的开头。

  • e → 到下一个单词的结尾。【w和e大小写时单词判断方式会不一样】

复制/拷贝

有一个功能键模板是这样的：\<start position>\<command>\<end position> ，这样能否实现自由拷贝呢？

还有N<command> → 重复某个命令N次

例如 0y$ 命令意味着：

  • 0 → 先到行头
  • y → 从这里开始拷贝
  • $ → 拷贝到本行最后一个字符

你也可以输入 ye，从当前位置拷贝到本单词的最后一个字符。

你也可以输入 y2/foo 来拷贝2个 “foo” 之间的字符串。

其中/pattern → 搜索 pattern 的字符串（陈皓注：如果搜索出多个匹配，可按n键到下一个）

超方便功能：(?是吗)

  • v (可视化的选择) → （陈皓注：可视化选择是一个很有意思的命令，你可以先按v，然后移动光标，你就会看到文本被选择，然后，你可能d，也可y，也可以变大写等）

自动补齐功能

Insert 模式下，输入单词开头，Ctrl + N 或 Ctrl + P 就可以自动补全啦~ 我个人倾向于 Ctrl + P ，另一个是浏览器快捷键。。。

可视化选择

先 ^ 移动到行头。

windows 下用Ctrl + Q ；也可直接键入v 或 V ，从光标所在位置开始，现在可以进行块操作了

向下移动光标选中各行。Ctrl + D 向下移动，方向键也可以

接下来：

  • < 或 > 来左右缩进，= 可以自动给缩进！
  • I<something> [ESC] → 插入something，ESC为每一行生效。
